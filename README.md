# TalkShow

A beautiful prompt sharing tool for AI coding agents. Generate stunning terminal-style cards for your prompts — like Carbon, but for AI commands.

## Features

- 🎨 **6 Beautiful Themes** — Nord, Dracula, Monokai, GitHub Dark, Vercel, Swiss
- 🖼️ **9 Gradient Backgrounds** — Plus solid colors and transparent option
- 🤖 **4 Agent Support** — Claude, OpenCode, Aider, Cursor
- ✨ **Export Options** — Copy command, copy image, or download PNG
- 🔧 **Customizable** — Font family, size, padding, window controls

## Run Locally

**Prerequisites:** Node.js 18+

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Tech Stack

- React 19
- Vite 6
- Tailwind CSS 4
- html-to-image
